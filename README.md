# Angular Learning Projects

This workspace contains 13 Angular applications covering various concepts from basics to advanced topics.

---

## Project Overview

| Project | Topic | When to Use |
|---------|-------|-------------|
| **app-1** | Angular Fundamentals | Learn the core building blocks: signals, conditionals, loops, switch-cases, event binding, and two-way data binding |
| **app-2** | Basic App Setup | See a minimal Angular application structure |
| **app-3** | Structural Directives | Understand how to use `*ngIf`, `*ngFor`, and custom structural directives |
| **app-4** | Component Communication | Learn parent-child component interaction and data sharing |
| **app-5** | State Management | Explore state management patterns with control-count, display-count, and store |
| **app-6** | Custom Pipes | Learn to create and use custom pipes for data transformation |
| **app-7** | Static Routing | See basic routing with static routes (about, contact, home, page-not-found) |
| **app-8** | Static Routing | Another routing example with docs, home, and profile pages |
| **app-9** | Dynamic Routing | Learn dynamic routing with route parameters and services |
| **app-10** | Static Routing | More routing examples with home, personal-details, profile, settings |
| **app-11** | User Details | Display and manage user details |
| **app-12** | Reactive Forms | Learn template-driven and reactive form handling in Angular |
| **app-13** | API Integration | Understand how to integrate REST APIs with Angular services |

## Running a Project

To run any project, navigate to its folder and serve:

```bash
cd app-X
ng serve
```

Then open `http://localhost:4200/` in your browser.
