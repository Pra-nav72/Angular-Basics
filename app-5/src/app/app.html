***** ***** GLOBAL STATE ***** *****

if want to share data b/w parent & child components, or among sibling components.
Then, we need a common place to store those data, this common place is called GLOBAL STATE

How to do:

  • 1. create a service (>>>ng generate service store/counter) [folder/file]

    • service is a file where we store commonly used data, elements, components etc

    counter.ts file will contain a @Injectable directive.

    which means it can be used inside any .ts file



  • 2. inside the counter.ts, we will create a signal and multiple functions
    to increase, decrease, & reset the value of the signal.


  • 3. we will use these function in different components

  • 4. create different components, & register the service (in constructor) to use

    • in component display count::: constructor(state: counter){}
    in the .html file of the same component:: {state.count()}, where count() is a signal of counter service

    similarly import funtions to run on button click which is defined in counter service

    Don't forget to add 'public' access modifier for state
